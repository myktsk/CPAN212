## Part 2: Book Store 📚

A bookstore website to test hosting a local server with nodejs.

### How to Run ✅

```sh
npm start
```

and open `http://localhost:3000` in your browser.

### Route Map 🌐

- `/` - Home
- `/about` - About
- `/book` - Book Detail
- `{invalid route}` - 404
